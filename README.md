PDF Conversion & Processing Toolkit
A Complete Document Transformation Engine using Spring Boot
Overview:
The PDF Conversion & Processing Toolkit is a powerful backend service built with Spring Boot,
providing unified REST APIs for converting, processing, optimizing, securing, splitting, merging, and
extracting data from PDF files.
Features:
- PDF → Word (DOCX) / Perfect Layout
- PDF → Text / HTML / Images
- Image → PDF / Word → PDF
- Merge, Split, Compress PDFs
- Watermarking
- Password Protect / Unlock
- Metadata Read/Write
Architecture:
Modular services with a single controller for clean API exposure.
Installation:
1. Clone repository
2. Build using Maven
3. Run Spring Boot application
